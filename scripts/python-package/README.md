# catbird-mls Python Bindings

Python bindings for the catbird-mls MLS (Messaging Layer Security) SDK.

## Installation

```bash
# Generate bindings first
cd catbird-mls
./scripts/generate-python-bindings.sh

# Install
cd build/python
pip install -e .
```

## Usage

```python
from catbird_mls import CatbirdMls

# See docs/cross-language-guide.md for full usage examples
```
